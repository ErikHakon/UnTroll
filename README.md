# ⚡ Anti-Troll (Rift Coach AI)

Coaching inteligente para League of Legends con Inteligencia Artificial.

La primera herramienta que analiza ambas composiciones con IA y te da builds, runas y estrategia adaptada a tu partida exacta.

## Features

- 🧠 **IA que razona** - No busca winrates genéricos, analiza POR QUÉ cada item es óptimo
- ⚔️ **Build de línea vs teamfight** - Separa la build en dos fases
- 🤝 **Sinergia de equipo** - Analiza a tus aliados para optimizar la build
- 🗺️ **Game Plan completo** - Estrategia por fases con tips contextuales
- 📊 **Análisis de daño** - Identifica la composición de daño enemiga

## Tech Stack

- React 18
- Vite
- Claude AI (Anthropic API)

## Setup

```bash
npm install
npm run dev
```
